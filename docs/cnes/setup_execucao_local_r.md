# Setup local para execução do pipeline CNES em R

## Objetivo

Descrever o preparo mínimo de ambiente para validar o pipeline piloto CNES fora do sandbox atual, incluindo instalação de dependências e execução de testes.

## Pré-requisitos

- R (recomendado: versão 4.3 ou superior)
- `Rscript` disponível no PATH
- Git

Verificação rápida:

    which R
    which Rscript
    R --version
    Rscript --version
